Displays Links to Essential WordPress references in the Toolbar. Designed for beginners and students!

== Description ==

* Cheat Sheets are only visible to logged in Administrators.
* Appears on the right-side of the Toolbar (admin bar). 

== Installation ==

1. Simply unzip and upload to the wp-content/plugins' directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. There's nothing to configure. Cheat Sheets will automatically appear when you view your website as an Administrator. Look at the upper-right corner of the Toolbar ("Admin Bar").
1. Hover over "Cheat Sheets" for a list of links


== Frequently Asked Questions ==

= I don't see Cheat Sheets! Where is it? =

You must be logged in as an administrator
